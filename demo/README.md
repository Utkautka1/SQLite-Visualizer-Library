# SQLite Visualizer Demo

This is a Next.js demo application showcasing the SQLite Visualizer library.

## Getting Started

First, install dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the demo.

## Features

- Load SQLite database files (.db, .sqlite, .sqlite3)
- Create new databases
- View ER diagrams
- Execute SQL queries
- Explore table data
- Analyze query plans
- View query history
